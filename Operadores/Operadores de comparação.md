# Operadores de comparação

## == (Comparação implícita)
```js
| Só compara o valor

const numero = 289;
const texto = "289";

console.log(numero == texto) // true
```

## === (Comparação explicíta)
```js
| Compara os valores e os tipos das variáveis

const numero = 904;
const texto = "904";

console.log(numero === texto) // false
```
