# String

```js
|| Tudo que for colocado entre aspas, é considerado uma string.
|| Pode ser aspas simples('') ou aspas duplas("").

const texto1 = "Olá, mundo!";
const texto2 = 'Olá, mundo!';
const senha = "senhaSuperSegura!";
const StringDeNumeros = "56754";
```

## Concatenação(+)

```js
|| Soma de strings

const citacao = "Meu nome é ";
const meuNome = "Kamila";

console.log(citacao + meuNome) //Meu nome é Kamila

```
